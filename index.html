What is Semantic tag :

Semantic tag hocche shei shob tag , jader k dekhle bujha jai oi tag er kaj ki / keno use kora hoyeche . kono ekta html doc e code gulo meaning full howar jonno & developer jathe code dekhei bujhte pare kothai ki modify hoyeche tar jonno semantic tag use kora hoi & code gulo k gochano vhabe arrange kora hoi .
